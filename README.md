# react-tic-tac-toe
A two-player Tic-Tac-Toe game made with React and local state management.

1. Clone the repo:

   ```bash
   git clone https://github.com/Cobb304/react-tic-tac-toe.git
   cd React-tic-tac-toe

2. Install dependencies:

   ```bash
   npm i

3. Run the development server:

   ```bash
   npm run dev
